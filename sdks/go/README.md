# GoudEngine Go SDK

Go wrapper for the GoudEngine game engine.

## Architecture

```
sdks/go/
  go.mod                    # Module: github.com/aram-devdocs/goud-engine-go
  goud/                     # Public wrapper package (generated by codegen)
    doc.go                  # Package documentation
    types.go                # Value types: Color, Vec2, Rect, Vec3, etc.
    components.go           # Component types: Transform2D, Sprite, etc.
    entity.go               # EntityID type
    keys.go                 # Key, MouseButton, and other enum constants
    errors.go               # GoudError and category types
    game.go                 # Game struct and all engine methods
  internal/ffi/             # Internal cgo bindings (generated by gen_go.py)
    doc.go                  # cgo flags (CFLAGS, LDFLAGS)
    ffi.go                  # Raw cgo wrappers for all C functions
    helpers.go              # Bridge functions for cross-package type safety
    ffi_test.go             # Nil pointer guard tests
  include/
    goud_engine.h           # C header (copied by codegen)
```

## Quick Start

```go
package main

import "github.com/aram-devdocs/goud-engine-go/goud"

func main() {
    game := goud.NewGame(800, 600, "My Game")
    defer game.Destroy()

    tex := game.LoadTexture("assets/player.png")

    for !game.ShouldClose() {
        game.BeginFrame(0, 0, 0, 1)

        if game.IsKeyJustPressed(goud.KeyEscape) {
            game.Close()
        }

        game.DrawSprite(tex, 100, 100, 64, 64, 0, goud.ColorWhite())
        game.EndFrame()
    }
}
```

## Building

The Go SDK requires the native GoudEngine library:

```bash
# Build the native library
cargo build --release

# Build Go code
cd sdks/go && CGO_ENABLED=1 go build ./...

# Run the Flappy Bird example
./dev.sh --sdk go --game flappy_bird
```

## Code Generation

All files in `goud/` and `internal/ffi/helpers.go` are auto-generated:

```bash
# Regenerate all Go SDK files
python3 codegen/gen_go_sdk.py    # Wrapper package
python3 codegen/gen_go.py        # Internal cgo bindings

# Or regenerate everything
./codegen.sh
```

Do not hand-edit generated files. Modify the generators in `codegen/go_codegen/` instead.
