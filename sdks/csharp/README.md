# GoudEngine C# SDK

[![NuGet](https://img.shields.io/nuget/v/GoudEngine.svg)](https://www.nuget.org/packages/GoudEngine/)

> **Alpha** -- This SDK is under active development. APIs change frequently. [Report issues](https://github.com/aram-devdocs/GoudEngine/issues)

Thin .NET 8.0 wrapper over GoudEngine's FFI layer via P/Invoke.

## Installation

```bash
dotnet add package GoudEngine
```

## Quick Start

```csharp
using GoudEngine;

var game = new GoudGame(800, 600, "My Game", RendererType.Renderer2D);

var tex = game.LoadTexture("assets/player.png");

game.Update(dt =>
{
    if (game.IsKeyPressed(Keys.Escape)) { game.Close(); return; }
    game.DrawSprite(tex, 400, 300, 64, 64);
});

game.Run();
```

## Architecture

This SDK is a thin wrapper -- all engine logic lives in Rust. `NativeMethods.g.cs` is auto-generated by csbindgen on `cargo build`. Do not hand-edit generated files.
